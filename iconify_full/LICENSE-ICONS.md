# Icon licenses (important)

**This package (`iconify_full`) is MIT licensed** — see [LICENSE](LICENSE).

**The icon artwork is not MIT.** Icons come from [Iconify Open Icon Sets](https://icon-sets.iconify.design/) ([iconify/icon-sets](https://github.com/iconify/icon-sets)). Each of the 200+ collections has its **own license** (Apache-2.0, MIT, CC BY 4.0, CC BY-SA, CC BY-NC, etc.).

## Your responsibility

If you ship an app that bundles icons (even via subsetting), **you** must:

1. **Check the license** for every icon set you use (prefix), on [icon-sets.iconify.design](https://icon-sets.iconify.design/).
2. **Follow attribution** requirements (many CC licenses require credit).
3. **Avoid restricted sets** for your use case (e.g. do not use **CC BY-NC** sets in commercial apps).
4. **Keep documentation** of which sets you ship if your legal/compliance team requires it.

`iconify_full` is a **tool** (download, subset, render). It does **not** grant rights to redistribute icon artwork.

## Example

| Icon id | Set prefix | Check license at |
|---------|------------|------------------|
| `mdi:home` | `mdi` | Material Design Icons — Apache-2.0 |
| `tabler:brand-flutter` | `tabler` | Tabler Icons — MIT |
| `arcticons:…` | `arcticons` | Arcticons — CC BY-SA 4.0 |

When in doubt, read the license on the icon set page before shipping.
